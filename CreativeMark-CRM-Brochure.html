What This Product Is

This is a Customer Relationship Management (CRM) system with integrated Enterprise Resource Planning (ERP) capabilities. It is designed for businesses that need to manage clients, leads, projects, finances, employees, partners, and field operations in one unified platform. The Android APK (CM Field) lets ground staff log visits, capture geo-tagged selfies, and share live location with managers.

CM Field — Android App Features

A 3.7 MB lightweight Capacitor-powered Android app for ground staff. Employees log in, start their shift with a selfie, and their manager sees them on the Live Tracking map instantly.

Download APK

File: app-debug.apk • Size: ~3.7 MB • Supports Android 8+ • Requires GPS & Camera

FREE

Secure Login

Employees log in with email + password via Supabase Auth. Auto-session management. Logout clears tracking.

Android

Selfie-Verified Shift Start/End

Employees take a selfie to start and end their shift. Photos are uploaded to Supabase Storage with timestamp + GPS coordinates. Prevents proxy attendance.

Android

Live GPS Heartbeat (Every 2 Min)

Location pushed to the server every 2 minutes while on shift. Manager sees live status: Live, Recent, Stale, or Offline.

Android

Society Visit Logging (Society One)

Log society name, address, contact person, phone, number of flats. Attach selfie + building photo. Auto-captures GPS and generates Google Maps link.

Android

Shop Visit Logging (Smart Tap AI)

Record shop owner details, mobile number, interest level (Interested / Not Interested / Follow Up), next call date, notes, and photos.

Android

My Assignments & Daily Tasks

Manager-assigned societies/tasks appear in the app daily. Mark assignments done with one tap. Pending vs Completed split view.

Android

Visit History & Edit

Review all past visits for today/week/month. Tap any visit to view details or edit society name, address, contact, and flat count.

Android

Auto-Sync & Real-Time Stats

Dashboard stats refresh every 30 seconds. Target vs visited progress bar. System status panel for GPS, permission, and tracking health.

Tech Stack & Architecture

Frontend

  • React 18 + TypeScript + Vite
  • Tailwind CSS + shadcn/ui components
  • Recharts for analytics
  • Leaflet + React-Leaflet for maps
  • React Query (TanStack) for data
  • React Router v6 for navigation

Backend & Data

  • Supabase (PostgreSQL + Auth + Realtime)
  • Row-Level Security (RLS) policies
  • Supabase Storage for photos & PDFs
  • Edge Functions for webhooks
  • Capacitor 6 for Android wrapper
  • jsPDF + autotable for PDF generation

Suggested Add-On Features for Growth

  • Invoice QR Code: Add UPI QR to generated PDFs for instant payment.
  • Multi-Language: Hindi/Marathi toggle for the field app to improve adoption.
  • Voice Notes: Let field staff record 30-sec voice memos instead of typing notes.
  • Expense Scanner: OCR receipt upload in the Financials module.
  • WhatsApp Business API: Replace web WhatsApp links with official Business API for automated drip campaigns.
  • Attendance Biometrics: Face-match verification between shift selfie and employee profile photo.
  • Customer Portal: Give clients a read-only view of their project progress and invoices.
  • Predictive Analytics: ML-based lead scoring to predict which leads will convert.
  • Multi-Branch: Add branch/territory scoping for franchise or multi-city operations.
